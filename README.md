## 💡 About the Project  
Estcourt Waste Collectors is a client project I developed as a web developer. The website highlights recycling, salvaged goods (including chicory), and community sustainability. It’s designed to be approachable, modern, and community‑focused, with clean navigation and eco‑friendly branding.  

---

## ✨ Features  
- ♻️ Clear navigation and responsive design  
- 🌍 Community‑focused messaging and sustainability goals  
- 👥 Salvaged goods marketplace with seasonal stock  
- 💡 Interactive elements: impact slider, counters, and call‑to‑action buttons  
- 📱 Mobile‑friendly layouts  

---

## 🛠 Tech Stack  
- **Frontend:** HTML5, CSS3, JavaScript  
- **Design:** Responsive grid layouts, eco‑friendly color palette  
- **Interactivity:** Smooth scrolling, dynamic counters, form handling
